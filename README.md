# Todo List App

![Flutter](https://img.shields.io/badge/Language-Dart-red.svg) 
![Status](https://img.shields.io/badge/Status-Complete-orange.svg)

A simple to-do list app built with Flutter, allowing users to manage their tasks in a clean and simple environment.

## Features

- Add, update, and delete tasks seamlessly.
- Tasks are sorted by completion status for easy organization.
- Intuitive design for a smooth user experience.
- Uses SQLite for local data persistence.

## Usage

1. Add a new task by tapping on the '+' button.
2. Swipe right on the switch to mark it as completed.
3. Long press on a task to delete it.
4. Enjoy organizing your tasks efficiently!

## Dependencies
The app uses the following dependencies:
- sqlflite: For storing data locally in a structured manner.
- shimmer: For enhancing the user interface with subtle loading animations.
